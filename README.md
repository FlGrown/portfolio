# Jorge R. Piquer — Portfolio

Personal portfolio site built with React and Vite. Features engineering case studies and an about page.

**Live site:** [jorgepiquer.com](https://jorgepiquer.com)

---

## Tech Stack

- React 19
- React Router v6
- TypeScript v6
- Vite
- CSS (no UI libraries)

## Running Locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run deploy
```

Deploys to the `gh-pages` branch which is served via GitHub Pages at [jorgepiquer.com](https://jorgepiquer.com).
